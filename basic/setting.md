# ⚙️ 提词设置

### 关于提词页面的相关设置

如果你需要调整提词的背景透明度、字体大小、滚动速度、段落间距、字体颜色、背景颜色等，可以在点击「开启提词」后，页面点击左下角的 ⚙️ 进行调整。
